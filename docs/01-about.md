# About

`phpsoftbox/queue` — лёгкий компонент очередей, рассчитанный на интеграцию с разными драйверами. В комплекте есть in-memory очередь и адаптер для БД.

Основные элементы:
- `QueueInterface` — контракт очереди
- `QueueJob` — модель задания с идентификатором и числом попыток
- `Worker` — обработчик задач с ретраями и callback на ошибку
- `DatabaseDriver` — хранение очереди в БД через компонент Database
- `FailedJobStoreInterface` — запись задач, которые исчерпали число попыток
